Overview

Auto translate Odoo data fields from preferred language to another chosen language and create bilingual printed documents. Reliable auto translate will enter the chosen language in available translation tables from preferred language and allow you to edit if necessary. Save time and cost by reducing manual translated text entry.

Features

Key Features Functionality
Auto-translated chosen language text via embedded Google translate feature Automate translation in Odoo tables will save time and simplify process


Refine entry if needed Flexibility for user to update translation in Odoo table
